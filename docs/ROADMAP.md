## ✅ Phase 1 — ХИЙГДСЭН (одоогийн систем)

| Easy Parking боломж | Манай систем |
|---|---|

| Үйлчилгээний хураамж тооцоолох (тарифын тохиргоо) | ✅ Тарифын загвар: шатлал, үнэгүй минут, grace, daily cap |
| Нэхэмжлэхийн модуль | ✅ Payments (QPAY/CASH/POS, статус хөтлөлт) |
| Хөнгөлөлтийн модуль (хувиар / цагаар) | ✅ PERCENT / FIXED / FREE_MINUTES |
| Үйлчилгээний вэбийн QR (дугаараар лавлаж төлөх) | ✅ /pay хуудас |
| Зогсоолын хаалтын QR (сүүлд танигдсан машинууд) | ✅ /pay recent-exits сонголт |
| Зогсоолын ажилтны төлбөр хураалт (бэлэн/карт/QR) | ✅ Касс + PAX POS API |
| НӨАТ-ийн баримт (иБаримт) | ✅ Автомат e-Barimt (mock, бодит холболт config-оор) |
| Админ портал: хэрэглэгч, эрх, зогсоол, хаалга, тариф | ✅ Тохиргоо + Хэрэглэгчид (RBAC 4 түвшин) |
| Зогсох эрх бүхий машины бүртгэл | ✅ Бүртгэлтэй жолооч (гэрээт/сарын) |
| Тайлангууд (төлбөр, лавлагаа, ашиглалт) | ✅ Тайлан + Excel экспорт |
| Хаалт нээх лавлагаа / аудит | ✅ Командын түүх + үйлдлийн лог |
| Үл мэдэгдэх нэвтрэлт/гаралт | ✅ EXIT_NO_SESSION alert + камерын event лог |
| Middleware (Dahua төхөөрөмжийн интеграц) | ✅ ITSAPI шууд + CGI barrier (edge PC шаардлагагүй хувилбар) |
| Хар жагсаалт | ✅ (Easy Parking-д байгаагүй — нэмэлт давуу тал) |
| Real-time хяналтын самбар (WebSocket) | ✅ (нэмэлт давуу тал) |

## 🔜 Phase 2 — Гуравдагч интеграц ба жолоочийн апп

Easy_System_2025-д байгаа, дараагийн шатанд хийх:

1. **Trd API (гуравдагч системийн API)** — TOKI, MostMoney, банкны аппуудад:
   - `addCustCar` — машин холбох
   - `getCarCurrent` — одоогийн зогсолт/төлбөр лавлах
   - `createInvoice` / `setInvoicePaid` — төлбөр
   - `getCarHistory` — түүх
   - `setOrgRegNo` — байгууллагын НӨАТ баримт
2. **Автомат төлбөр** — хэтэвчээс гарах үед автоматаар суутгах (webhook бүртгэл)
3. **Жолоочийн mobile апп** — сул зогсоол газрын зураг, олон машин, авто төлбөр,
   түүх, НӨАТ баримт ([MOBILE_APP_GUIDE.md](MOBILE_APP_GUIDE.md)-ийн Phase 2 хэсэг)
4. **Үйлчилгээний байгууллагын хөнгөлөлт** — байгууллага бүртгэх, лимит,
   Used/UnUsed купон, байгууллагын ажилтны эрх
5. **Цаг захиалга** — урьдчилж байр захиалах (нэмэлт төлбөртэй)
6. **Цагийн хуваарьт тариф** — ажлын цаг / орой / амралтын өдрөөр ялгаатай үнэ
7. **Excel импорт** — гэрээт жолоочийг файлаар оруулах
8. **Push мэдэгдэл** — хугацаа дуусах сануулга гэх мэт

**Phase 2-т үлдсэн (App/Admin/JGA табуудаас):**
- Жолоочийн апп: газрын зурган дээр zогсоолын pin + сул тоо, booking (урьдчилсан захиалга,
  цуцлалтын логик), данс цэнэглэх хэтэвч (Hipay загвар), push notification, OTP бүртгэл,
  нууц үг сэргээх, олон машин холбох
- Snapshot: камерын зургийг session-д хавсаргаж дугаар засахад харуулах, 90 хоног хадгалах
- Dynamic permission: хэрэглэгч бүрд цонх + CRUD эрх тохируулах (одоо 4 роль-оор)
- Дүүргэлтийн хасалт/тооллого, багана сонгодог тайлан (A4 хэвлэлт)
- Хөнгөлөлтийг Excel-ээр олноор импортлох, хөнгөлөлтийн товчилсон нэр (ХБИ г.м)
- Кассын ажилтанд зогсоол дүүрч буй мэдэгдэл (80% шар / 90% улаан)
- ТОКИ систем рүү мэдээлэл дамжуулах, ETAX
